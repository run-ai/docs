## Description

Show list of Nodes (machines), their capacity and utilization.

## Synopsis

    runai top node 
        [--help | -h]
        [--details | -d]

## Options


### Global Flags

#### --loglevel (string)

>  Set the logging level. One of: debug | info | warn | error (default "info").

#### --help | -h

>  Show help text.

#### --details | -d

>  Show additional details.

## Output

Shows a list of Nodes their capacity and utilization.
