## Description

Log out from Run:AI

## Synopsis

``` shell
runai logout 
    [--loglevel value]
    [--help | -h]
```

## Options

--all | -A
>  Resume all suspended Jobs in the current Project.

### Global Flags

--loglevel (string)
>  Set the logging level. One of: debug | info | warn | error (default "info").

--help | -h
>  Show help text.

## Output

You will be logged out from Run:AI
