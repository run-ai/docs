# Uninstall Run:AI 


## Uninstall a Run:AI Cluster
To uninstall the cluster see: [cluster delete](../../cluster-setup/cluster-delete.md) 


## Uninstall the Run:AI Backend

To delete the backend, run:

``` shell
helm delete runai-backend -n runai-backend

```
