## Description

Log out from Run:ai

## Synopsis

``` shell
runai logout 
    [--loglevel value]
    [--help | -h]
```

## Options

### Global Flags

#### --loglevel (string)
>  Set the logging level. One of: debug | info | warn | error (default "info").

#### --help | -h
>  Show help text.

## Output

You will be logged out from Run:ai
